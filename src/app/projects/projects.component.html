Tilty Dude

Tilt to win a new high score! Tilty Dude is a video game with a side-to-side tilting controller and dangerous falling objects; tilt the dude left or right to avoid these objects and win. This system is made to operate using a PSoC 5LP stick, a TFT screen to display the game, and an MPU6050 accelerometer as the main controller.

Tilty Dude was an individual final project for 6.115, Microcomputer Project Lab during Spring 2022.

Mr. Cat

Mr. Cat is a quick game I made during Spring Break 2022 out of boredom. I was staying in Boston and needed something to do. While the game itself is pretty simple, it's somehow enticing. Mr. Cat was made with Unity Engines, written mostly in C#. Play full screen to see the score.

First Light

First Light is a game made during a one-month website creation competition hosted by MIT. It is a 2D pixelated personality-based website game encouraging players to learn more about themselves. We added character personalization functionalities with a secure login system (Google) and data storage capabilities. Players can re-log in, save nicknames, and view their progress throughout the game. This website is built with React and MongoDB.

Finger Dance Dance Revolution

Finger Dance Dance Revolution is a final project made by my small group in an Interconnected Embedded Systems class. It was made using an ESP32 microcontroller, joystick shield, speaker, and LCD screen, with minimal HTML and CSS site coding, and loads of C++.

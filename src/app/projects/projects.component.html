First Light

First Light is a game made by my friends Sofia, Erika and me during a one-month website creation competition hosted by our school. It is a 2D pixelated personality-based website game encouraging players to learn more about themselves. We added character personalization functionalities with a secure login system (Google) and data storage capabilities. Players can re-log in, save nicknames, and view their process throughout the game. This website is built using React and MongoDB for our backend.

Finger Dance Dance Revolution

Finger Dance Dance Revolution is a final project made by my small group in an Interconnected Embedded Systems class. It was made using an ESP32 microcontroller, joystick shield, speaker, LCD screen, minimal HTML and CSS site coding, and loads of C++.
